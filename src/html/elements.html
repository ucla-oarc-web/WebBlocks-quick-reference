Base Elements

Foundational elements

Container

Container with maximum width and minimum gutter

<div class="container"><!-- content --></div>

Row

X/12 proportional above collapse; full width below

<div class="row">
    <div class="panel-4"><!-- 1/3 --></div>
    <div class="panel-8"><!-- 2/3 --></div>
</div>

Right-oriented Row

Right-to-left orientation above collapse; DOM order below

<div class="row right">
    <div class="panel-4"><!-- right --></div>
    <div class="panel-8"><!-- left --></div>
</div>

Visibility

To completely hide content

<div class="hide">

To hide content except for screen readers

<div class="hide-accessible">

To hide content except if no media query support

<div class="hide-media-query">

Block Float

For X as left or right, use float-X

<div class="float-right">

Text Alignment

For X as left, center, right, justify, use text-X

<div class="text-right">

Responsive Breakpoints

medium-small - 768px

Responsive Row Collapse

<div class="row medium-collapse">

Responsive Visibility

For X breakpoint, use hide-X to hide below X

<div class="hide-medium">

For X breakpoint, use hide-above-X to hide above X

<div class="hide-above-medium">

Responsive Block Float

For X breakpoint, Y as left or right...

Use float-Y-X to float Y below X

<div class="float-right-medium">

Use float-Y-above-X to float Y above X

<div class="float-right-above-medium">

Responsive Text Alignment

For X breakpoint, Y as left, center, right, justify...

Use text-Y-X to align text to Y below X

<div class="text-center-medium">

Use text-Y-above-X to align text to Y above X

<div class="text-right-above-medium">

Colors

Colors can be applied on any element, and some contexts have special behaviors.

Classes

primary, secondary, tertiary, neutral, default, success, warning, error, danger, important, highlight, info, inverse

Modifiers

Modifiers can be applied to any color to create variation:

gradient - gradient variant of base color
dark - dark variant of base color
light - light variant of base color
fade - dark if body background dark, light otherwise
emphasize - light if body background dark, dark otherwise

Any brightness modifier may be used with gradient

Special Behaviors

.button implicitly has gradient
.message implicitly has fade
.control colors label, input border, help text

Text Classes

The text- classes provide only text color, and they do not have support for modifiers.

<span class="text-primary"><!-- primary --></span>

Constrained Content

Use constrained to avoid overflowing containing element

<img class="constrained">
